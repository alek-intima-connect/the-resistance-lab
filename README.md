# The Resistance Lab

> "We feed on the old world to nurture the new."

**The Resistance Lab** is the digital command center for a proletariat-funded network of sanctuaries. It serves as a repository for historical truths, a coordination hub for mutual aid, and a funding engine for community infrastructure.

## /// SYSTEM STATUS
- **Version:** 1.0.0 (Alpha)
- **Architecture:** Static HTML/Tailwind served via Node.js
- **Network:** Decentralized / Local-First
